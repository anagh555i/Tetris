Objective

The goal of Tetris is to clear lines by strategically placing falling tetrominoes (shapes made of four blocks) into a well or matrix.

Controls

Left/Right: Use the left and right arrow keys (or corresponding keys on your device) to move the tetromino left or right.
Rotate: Use the up arrow key to rotate the tetromino. Each press rotates the shape 90 degrees.
Soft Drop: Use the down arrow key to speed up the fall of the tetromino.
Hard Drop: Use the Enter key to instantly drop the tetromino to the bottom.
Soft Drop : 1 x Distance
Hard Drop : 2 x Distance
Single Line Clear : 100
Double Line Clear : 200
Triple Line Clear : 400
Quadruple Line Clear: 800
